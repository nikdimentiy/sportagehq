# 🚗 Sportage HQ Command Center

A futuristic command center for tracking your Kia Sportage across fuel, mileage, maintenance, and backup workflows — all in one sleek dashboard. It combines real-time analytics, structured logging, and cloud sync into a single web app experience.

## ✨ Overview

Sportage HQ is a dark, sci-fi–styled vehicle tracking dashboard built around five core systems: Overview, FuelOS, MileageOS, Maintenance, and Data Vault. It includes authentication, cloud sync, charts, reports, CSV import, JSON backup/export, and a responsive interface designed for fast daily use.

## 🧩 Core Modules

### 📊 Overview
A high-level dashboard showing total fuel spend, current odometer, lifetime average MPG, fuel logs, mileage logs, gallons used, cost per mile, days tracked, and trend charts such as monthly fuel spend and MPG trend.

### ⛽ FuelOS
Track every fuel stop with date, station, grade, gallons, price per gallon, trip distance, odometer, total cost, MPG, and cost per mile. FuelOS also supports CSV import, monthly reports, peak activity insights, export options, and reset controls.

### 🛣️ MileageOS
Log odometer entries with timestamp support and analyze driving behavior through weekly mileage, last trip, weekly average, average trip, longest run, busiest day, total driven distance, and daily average metrics.

### 🔧 Maintenance
Store service records including date, service type, notes, shop, odometer, cost, next due date, and next due mileage. The maintenance section also summarizes total services, total spent, last service date, and last recorded odometer value.

## 🚀 Features

- Login-protected command center with account-aware navigation  
- Real-time cloud integration for auth, databases, and sync  
- Live charts for spend and MPG trends

- CSV import for fuel logs and JSON import/export for backups

- Responsive layout optimized for desktop and smaller screens
